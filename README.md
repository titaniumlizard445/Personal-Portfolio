# Personal-Portfolio
Contains a variety of programming projects that have been created using python
***
## How to Use
1. Click on the src folder
2. Click on the main.py file
3. Click on the triangle at the top right of the screen/run button
4. Select project
5. Click the Run button
6. If prompted, exit the windown and go to the terminal (don't close the page)
***
## Projects included:
- Fractal Generator
- Video Game Character manager
- Text-Based Adventure Game
- Turtle Race Game
***
## Libraries to install
- CustomTkinter
- Faker
- Matplotlib
- Pandas
